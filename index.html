เชื่อมต่อ Google Sheets

ใช้ Google Sheet เป็นฐานข้อมูลกลาง — ทีมช่วยกันกรอกในชีต แผนที่ดึงมาแสดงให้ทันที (อ่านอย่างเดียว กด “รีเฟรช” เพื่ออัปเดต)

  1. สร้างชีตที่มีหัวคอลัมน์ตามนี้ (แท็บแรก):
level, policy, code, name, lat, lng, installed, needed, status, problem, solution, note
  1. level = district / khwaeng / point
  2. code = รหัสเขต (เช่น 1033) หรือรหัสแขวง (เช่น 103301); สำหรับ point ใส่รหัสเขตที่หมุดนั้นอยู่
  3. status = done/prog/short/none (หรือไทย: ครบ/กำลัง/ขาด/ยังไม่)
  4. ที่ Google Sheets เลือก ไฟล์ → แชร์ → เผยแพร่ไปยังเว็บ → ทั้งเอกสาร → CSV แล้วคัดลอกลิงก์
  5. วางลิงก์ CSV ด้านล่าง แล้วกด “โหลด”

หมายเหตุ: ถ้าเปิดไฟล์นี้แบบ file:// บางเบราว์เซอร์อาจบล็อกการดึงข้อมูลข้ามเว็บ — แนะนำวางไฟล์บนโฮสติ้ง (GitHub Pages ฯลฯ) หรือใช้โหมดในเครื่อง + นำเข้า/ส่งออก
